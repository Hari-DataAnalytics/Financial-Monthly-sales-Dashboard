# Financial-Monthly-sales-Dashboard
Excel-based Sales Dashboard to analyze revenue, product performance, and regional trends using Pivot Tables, Slicers, and Charts.

 Financial Sales Performance Dashboard (Excel)

 Project Overview
This project is an interactive Sales Dashboard built using Microsoft Excel to analyze and visualize business performance data. It helps in understanding sales trends across different regions, product categories, and customer segments.

 Objectives
Analyze sales performance across regions and products
Track key performance indicators (KPIs)
Identify trends and patterns in sales data
Provide insights for better decision-making

 Tools & Technologies
Microsoft Excel
Pivot Tables
Slicers
Charts & Graphs

 Key Features
Interactive dashboard with slicers for filtering
Region-wise and country-wise sales analysis
Product and segment performance tracking
Clear visualization of revenue trends

 KPIs Tracked
Total Revenue
Sales by Region
Sales by Product
Segment-wise Performance

 Conclusion

This project showcases the use of Excel for data analysis and dashboard creation, helping to transform raw data into meaningful insights for business decisions.
